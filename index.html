JustJs

JustJs is a set of common heavily-used utilities/helpers for javascript environments. The goal is to collect the most used code throughout developments in javascript without compromising bundle size.

Usage

JustJs is available on npm for server usage and distributed over a CDN for browser usage.

Browser

To use the latest version of JustJs' core over a CDN, include the following anywhere on your HTML:

<script src='https://cdn.jsdelivr.net/npm/justjs@v1.0.0/dist/browser/core.min.js' async></script>

Then, if you want, you can replace "v1.0.0" with any of these versions and/or "core.min.js" with any of these files.

Available flavors

JustJs releases multiple bundles under the same repository, so that some bundles might not be available on older versions. For example: "basic.min.js" won't be available until v1.x.0, but you can use "core.min.js" since v1.0.0.

History

JustJs was initially called APR and it was born as a close-source project some years ago. —When IE8 was a thing and you had to decide whether to still targeting IE6 or go straight into IE8.

The main goal of APR was to ship it together as a suite (APR-suite) within every site I'd developed. However, I let that idea vanish away and I decided to rename the project to JustJs and released it as an open-source project on Github to let other people use it. —In fact, APR-suite is still being renamed, rewritten, and added to JustJs.

Now, after some time, I'm proud to say that JustJs is still the most used library throughout my developments and it's being more active than ever...
